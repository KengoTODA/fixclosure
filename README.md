fixclosure
====
fixclosure is JavaScirpt linter/fixer based on Esprima for Google Closure Library.
now alpha version...

## Install

```bash
$ npm install -g fixclosure
```

## Usage

```bash
# Lint
$ fixclosure foo.js

# Lint & Fix in place
$ fixclosure -i foo.js
```
